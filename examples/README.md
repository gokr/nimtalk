# Nemo Examples

This directory contains example programs demonstrating Nemo features.

## Running Examples

```bash
# Run an example
nemo 01_hello.nemo

# Or with debug output
nemo --loglevel DEBUG 02_arithmetic.nemo

# Show AST before execution
nemo --ast 05_classes.nemo
```

## Example Overview

| File | Description |
|------|-------------|
| `01_hello.nemo` | Basic "Hello, World!" program |
| `02_arithmetic.nemo` | Numbers and arithmetic operations |
| `03_variables.nemo` | Variable assignment and usage |
| `04_objects.nemo` | Basic object creation and property access |
| `05_classes.nemo` | Class creation with instance variables |
| `06_methods.nemo` | Method definition and calling |
| `07_inheritance.nemo` | Inheritance and super sends |
| `08_collections.nemo` | Arrays and Tables |
| `09_control_flow.nemo` | Conditionals and loops |
| `10_blocks.nemo` | Blocks and closures |
| `11_fibonacci.nemo` | Fibonacci calculation (recursive and iterative) |
| `12_stdlib.nemo` | Complete standard library demonstration |

## Quick Start

Start with the first few examples to learn the basics:

```bash
nemo 01_hello.nemo
nemo 02_arithmetic.nemo
nemo 05_classes.nemo
```

Then explore more advanced features:

```bash
nemo 07_inheritance.nemo
nemo 11_fibonacci.nemo
nemo 12_stdlib.nemo
```
