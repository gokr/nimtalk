# Nimtalk Examples

This directory contains example programs demonstrating Nimtalk features.

## Running Examples

```bash
# Run an example
ntalk 01_hello.nt

# Or with debug output
ntalk --loglevel DEBUG 02_arithmetic.nt

# Show AST before execution
ntalk --ast 05_classes.nt
```

## Example Overview

| File | Description |
|------|-------------|
| `01_hello.nt` | Basic "Hello, World!" program |
| `02_arithmetic.nt` | Numbers and arithmetic operations |
| `03_variables.nt` | Variable assignment and usage |
| `04_objects.nt` | Basic object creation and property access |
| `05_classes.nt` | Class creation with instance variables |
| `06_methods.nt` | Method definition and calling |
| `07_inheritance.nt` | Inheritance and super sends |
| `08_collections.nt` | Arrays and Tables |
| `09_control_flow.nt` | Conditionals and loops |
| `10_blocks.nt` | Blocks and closures |
| `11_fibonacci.nt` | Fibonacci calculation (recursive and iterative) |
| `12_stdlib.nt` | Complete standard library demonstration |

## Quick Start

Start with the first few examples to learn the basics:

```bash
ntalk 01_hello.nt
ntalk 02_arithmetic.nt
ntalk 05_classes.nt
```

Then explore more advanced features:

```bash
ntalk 07_inheritance.nt
ntalk 11_fibonacci.nt
ntalk 12_stdlib.nt
```
