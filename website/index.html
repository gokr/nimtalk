Nemo is a prototype-based Smalltalk dialect that compiles to native code. File-based, git-friendly, and designed for the modern era.

Get Started See Examples
example.nemo 
"Define a Point class with x and y slots"
Point := Object derive: #(x y)

"Add methods using the >> syntax"
Point extend: [
    self >> moveBy: dx and: dy [
        x := x + dx
        y := y + dy
        ^ self
    ]

    self >> distanceFromOrigin [
        ^ ((x * x) + (y * y)) sqrt
    ]
]

"Create and use a Point"
p := Point new
p x: 100 y: 200
p distanceFromOrigin println

Features

Everything you love about Smalltalk,
reimagined for today.

Prototype-Based Objects

Class-based with prototype inheritance. Create objects, derive new ones, and build your object model dynamically.

Native Compilation

Compiles to Nim, which compiles to C, which compiles to machine code. Fast, efficient, no VM required.

File-Based & Git-Friendly

No image files. Source code lives in .nemo files you can version control, diff, and edit with any editor.

Multiple Inheritance

Full support for multiple inheritance with conflict detection and resolution. Compose behaviors, don't inherit monoliths.

Green Threads

Cooperative multitasking with first-class Process objects. Built-in scheduler with round-robin execution.

Full Nim Interop

Call Nim code directly via FFI. Access the entire Nim ecosystem: libraries, packages, and system APIs.

Examples

Beautiful, expressive code.

Define a class 
"Create a Person class with name and age slots"
Person := Object derive: #(name age)

"Add instance methods"
Person extend: [
    self >> greet [
        ^ "Hello, I am " , name
    ]

    self >> haveBirthday [
        age := age + 1
        ^ self
    ]
]

"Class-side method for convenient creation"
Person extendClass: [
    self >> newNamed: n aged: a [
        | person |
        person := self derive
        person name: n
        person age: a
        ^ person
    ]
]
Use it 
"Create instances"
alice := Person newNamed: "Alice" aged: 30
bob := Person newNamed: "Bob" aged: 25

"Send messages"
alice greet println
"Alice is " , (alice age asString) println

"Mutate state"
alice haveBirthday
"Now Alice is " , (alice age asString) println
Block closures 
"Simple block evaluation"
result := [ 1 + 2 ] value

"Block with parameters"
double := [ :x | x * 2 ]
double value: 5  "Returns 10"

"Block with temporaries"
compute := [ :n |
    | squared doubled |
    squared := n * n
    doubled := n + n
    squared + doubled
]
compute value: 5  "Returns 35"
Control flow 
"Conditionals are message sends"
(x > 0)
    ifTrue: [ "positive" println ]
    ifFalse: [ "negative" println ]

"While loops"
count := 0
[ count < 10 ] whileTrue: [
    count println
    count := count + 1
]

"Non-local return (^) from block"
findPositive := [ :arr |
    arr do: [ :n |
        (n > 0) ifTrue: [ ^ n ]
    ].
    ^ nil
]
Green threads 
"Fork a new process"
worker := Processor fork: [
    1 to: 10 do: [ :i |
        Stdout writeline: i
        Processor yield
    ]
]

"Process control"
worker suspend
worker resume
worker terminate

"Run scheduler until completion"
Scheduler runToCompletion: 1000
Shared state 
"Globals for inter-process communication"
Nemo at: "counter" put: 0

"Two processes updating shared counter"
p1 := Processor fork: [
    1 to: 5 do: [ :i |
        current := Nemo at: "counter"
        Nemo at: "counter" put: (current + 1)
        Processor yield
    ]
]

p2 := Processor fork: [
    1 to: 5 do: [ :i |
        current := Nemo at: "counter"
        Nemo at: "counter" put: (current + 1)
        Processor yield
    ]
]
Arrays and Tables 
"Array literal - ordered collection"
numbers := #(1 2 3 4 5)

"Table literal - key-value dictionary"
scores := #{
    "Alice" -> 95
    "Bob" -> 87
    "Carol" -> 92
}

"Object literal - creates an object with slots"
config := {| host: "localhost" port: 8080 |}
Collection methods 
"Iterate with do:"
numbers do: [ :each | each println ]

"Transform with collect:"
squares := numbers collect: [ :n | n * n ]

"Filter with select:"
evens := numbers select: [ :n | (n \ 2) = 0 ]

"Reduce with inject:into:"
sum := numbers inject: 0 into: [ :acc :n | acc + n ]

"Find first match"
firstEven := numbers detect: [ :n | (n \ 2) = 0 ]

Get Started

Install Nemo.

Build from Source

git clone https://github.com/gokr/nemo.git
cd nemo
nimble local

Requires Nim 2.0+ and nimble.

Quick Start

nemo                           # Start REPL
nemo script.nemo               # Run a script
nemoc compile prog.nemo        # Compile to binary

Use --loglevel DEBUG for verbose output.
